# Shieldnet
> Extensão de navegador para bloqueio inteligente de conteúdos tóxicos.

##  Detalhes Técnicos da Sprint 1

* **Interface Moderna:** Uso de CSS avançado com *flexbox* e *blur effects* para uma experiência de usuário limpa e segura.
* **Lógica de Estado:** Implementação de lógica em JavaScript para gerenciar o status da proteção e contagem de bloqueios em tempo real.
* **Gestão Ágil:** Desenvolvimento guiado por *Product Backlog* e *User Stories*, garantindo que cada linha de código atenda a uma necessidade real da usuária.
